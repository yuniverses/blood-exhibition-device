# 血型展覽貼紙系統

展覽追蹤系統的前端客戶端，包含iPad輸入介面與大螢幕展示系統。

## 系統架構

- **輸入端 (iPad)**: 訪客輸入姓名和血型資料
- **展示端 (大螢幕)**: 顯示血型動畫和統計資訊
- **貼紙列印**: 生成包含QR Code的個人化貼紙
- **即時同步**: 透過WebSocket實現雙螢幕即時連動

## 安裝步驟

1. 安裝依賴套件：
```bash
npm install
```

2. 設定環境變數：
```bash
cp .env.example .env
```
編輯 `.env` 檔案，將 `API_BASE_URL` 改為實際的API位址

3. 啟動服務：
```bash
npm start
```

開發模式：
```bash
npm run dev
```

## 存取頁面

- 輸入頁面 (iPad): `http://localhost:8080/input`
- 顯示頁面 (大螢幕): `http://localhost:8080/display`

## 系統功能

### 輸入頁面特色
- 適配iPad觸控操作
- 直覺式血型選擇介面
- 即時表單驗證
- QR Code即時生成
- 貼紙預覽與列印

### 展示頁面特色
- 待機動畫循環播放
- 個人化歡迎訊息
- 血型知識動態展示
- 統計資料視覺化
- 自動返回待機模式

### 貼紙格式
- 訪客姓名
- 血型標示
- 唯一識別QR Code
- 適合標準貼紙列印尺寸

## 血型資料

系統包含8種血型的詳細資訊：
- A+, A-, B+, B-, AB+, AB-, O+, O-

每種血型都有：
- 科普介紹
- 台灣人口比例
- 視覺化配色

## 部署注意事項

1. 確保API服務正常運行
2. 檢查網路連線（內網環境）
3. iPad瀏覽器設定為全螢幕模式
4. 大螢幕設定適當的顯示比例
5. 配置列印機驅動程式

## 技術棧

- Node.js + Express
- Socket.io (即時通訊)
- QRCode.js (QR碼生成)
- 原生JavaScript (前端)
- CSS3動畫

## API整合

系統連接到展覽追蹤API，支援：
- 用戶註冊
- 資料查詢
- 互動記錄
- 統計分析

## 故障排除

### 無法連接API
- 檢查 `.env` 中的 `API_BASE_URL`
- 確認API服務運行中
- 檢查防火牆設定

### 列印問題
- 確認瀏覽器列印權限
- 檢查列印機連線
- 調整列印設定中的縮放比例

### 顯示異常
- 清除瀏覽器快取
- 檢查螢幕解析度設定
- 確認JavaScript已啟用
