# 影片檔案目錄

請將以下影片檔案放置於此目錄：

## 任務情境影片

1. `emergency_accident.mp4` - 車禍急救情境
2. `emergency_surgery.mp4` - 手術用血情境
3. `emergency_cancer.mp4` - 白血病治療情境
4. `emergency_trauma.mp4` - 大量失血情境
5. `emergency_birth.mp4` - 產後大出血情境
6. `emergency_heart.mp4` - 心臟手術情境

## 結果影片

1. `success.mp4` - 血型符合成功影片
2. `failed.mp4` - 血型不合失敗影片

## 預設影片

1. `idle.mp4` - 待機狀態循環播放影片

## 影片規格建議

- 解析度：1920x1080 (Full HD) 或更高
- 格式：MP4 (H.264 編碼)
- 長度：
  - 任務情境影片：10-20秒
  - 結果影片：5-8秒
  - 預設影片：30-60秒（循環播放）
- 音訊：可選（建議關閉或低音量）

## 注意事項

- 影片檔案名稱必須完全符合上述名稱
- 請確保影片檔案大小適中，避免載入過慢
- 影片內容應簡潔明瞭，配合文字疊加效果
