# 展場控制系統 - 螢幕選擇功能說明

## 📺 功能說明

Exhibition Controller 已經內建**螢幕選擇功能**，可以讓您選擇每個裝置畫面要顯示在哪個螢幕/投影機上。

---

## 🎯 使用方式

### 1. 位置

在每個裝置卡片中，您會看到：

```
┌─────────────────────────────────────┐
│ 裝置名稱                            │
├─────────────────────────────────────┤
│                                     │
│ 畫面列表：                          │
│   ┌──────────────────────────────┐ │
│   │ iPad 輸入頁面                │ │
│   │                              │ │
│   │ [顯示器 0 (主)▼]  [開啟]   │ │
│   └──────────────────────────────┘ │
│   ┌──────────────────────────────┐ │
│   │ 大螢幕展示                   │ │
│   │                              │ │
│   │ [顯示器 1     ▼]  [開啟]   │ │
│   └──────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

### 2. 螢幕選擇下拉選單

**位置**: 每個畫面項目的「開啟」按鈕左側

**外觀**:
- 紫色漸層背景
- 較粗的紫色邊框
- 滑鼠移上去會有陰影效果
- 聚焦時會有金色光暈

**選項**:
- `顯示器 0 (主)` - 您的主螢幕
- `顯示器 1` - 第二個螢幕/投影機
- `顯示器 2` - 第三個螢幕/投影機
- ...等

---

## 🎬 操作步驟

### 開啟裝置畫面到指定螢幕

1. **啟動裝置**
   - 點擊裝置卡片下方的「啟動裝置」按鈕
   - 等待裝置狀態變成綠色 (running)

2. **選擇螢幕**
   - 找到要開啟的畫面（例如：「大螢幕展示」）
   - 點擊該畫面左側的下拉選單
   - 選擇目標螢幕編號

3. **開啟畫面**
   - 點擊「開啟」按鈕
   - 畫面會在選定的螢幕上全螢幕顯示

### 移動已開啟的畫面

如果畫面已經開啟，想移動到另一個螢幕：

1. 向下捲動到「開啟的視窗」區域
2. 找到對應的視窗
3. 使用「移至顯示器」下拉選單選擇新螢幕
4. 視窗會自動移動到新螢幕

## 🔧 配置文件設定

在 `devices.config.json` 中，您可以為每個畫面設定預設螢幕：

```json
{
  "screens": [
    {
      "id": "input",
      "name": "iPad 輸入頁面",
      "url": "/input",
      "defaultDisplay": 0  // 預設在主螢幕
    },
    {
      "id": "display",
      "name": "大螢幕展示",
      "url": "/display",
      "defaultDisplay": 1  // 預設在第二螢幕
    }
  ]
}
```

## ❓ 常見問題

### Q: 為什麼我只看到一個螢幕選項？

**A**: 這表示系統只偵測到一個螢幕。請確認：
- 第二個螢幕/投影機已連接
- 螢幕已開啟
- 在系統偏好設定中可以看到多個顯示器

### Q: 下拉選單看不清楚怎麼辦？

**A**: 更新後的樣式使用了明顯的紫色漸層和陰影效果。如果還是看不清，可能需要：
1. 重新啟動 Exhibition Controller
2. 檢查顯示器亮度設定
3. 確認沒有顏色反轉或高對比度模式

### Q: 可以在開啟後移動視窗嗎？

**A**: 可以！向下捲動到「開啟的視窗」區域，使用那裡的「移至顯示器」功能。

### Q: 配置文件中的 defaultDisplay 有什麼用？

**A**: 這會設定下拉選單的預設選項。當您經常需要將某個畫面開在特定螢幕時很有用。
